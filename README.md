## вңЁ Overview

**Mastermind** solves the fragmented problem of career management and financial tracking by bringing everything together in one intelligent platform. Whether you're hunting for your next role, building a standout resume, watching your investments, or keeping tabs on your spending вҖ” Mastermind has you covered.

### рҹҺҜ Core Capabilities at a Glance

| Module | What it does |
|--------|-------------|
| рҹ”җ **Auth System** | Secure login / signup / reset via Supabase Auth + MongoDB user profiles |
| рҹ’ј **Job Search** | Live job listings via Adzuna + JSearch APIs with skill-matched recommendations |
| рҹ“„ **Resume Builder** | AI-generated, ATS-friendly resumes with PDF export powered by Gemini 2.0 Flash |
| рҹ“Ҳ **Market Dashboard** | Real-time stock quotes, charts, watchlist & portfolio tracking via Finnhub + Alpha Vantage |
| рҹ“° **News Feed** | Personalized tech/career news via NewsAPI with sentiment-tagged articles |
| рҹ’° **Expense Tracker** | Full income/expense management with category filters and financial snapshots |
| рҹӨ– **AI Chatbot** | Persistent Gemini-powered assistant with conversation history |
| рҹҺҜ **Daily Goals** | Localhost-persisted goal tracker for daily productivity |

```bash
# 1. Clone the repository
git clone https://github.com/your-username/mastermind.git

npm install
```

```bash
npm run build
npm start
```
